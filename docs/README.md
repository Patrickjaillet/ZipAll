# docs

Holds documentation assets referenced from the root [`README.md`](../README.md).

- **`screenshot.png`** (not yet present) — a screenshot of the main window,
  ideally mid-compression with the exclusion list populated. WinForms only
  renders on Windows, so this must be captured on a real Windows 10/11
  desktop session (or VM) running a built copy of ZipAll; it can't be
  produced from this project's Linux-based authoring/build environment.
  Once captured, save it here as `screenshot.png` and it will show up in the
  root README automatically.
